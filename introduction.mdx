## Overview

Self-comp empowers you to create, organize, and maintain project documentation in one centralized space. You build structured knowledge bases with version control, search, and collaboration features tailored for teams. Streamline your workflows by turning scattered notes into professional docs accessible to everyone.

<Callout kind="info">
  Self-comp integrates seamlessly with your existing tools like GitHub and Slack for automated updates.
</Callout>

## Quick Start

Get up and running in minutes:

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your account at https://dashboard.example.com/signup.
  </Step>
  <Step title="Create Project" icon="plus">
    Start a new documentation space for your project.
  </Step>
  <Step title="Add Content" icon="edit-3">
    Upload Markdown files or use the visual editor.
  </Step>
  <Step title="Publish" icon="globe">
    Share your live docs with your team.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Fetch your docs via API
  const response = await fetch('https://api.example.com/v1/docs/YOUR_PROJECT_ID', {
    headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
  });
  const docs = await response.json();
  console.log(docs);
  ```
  ```python
  # Fetch your docs via API
  import requests
  response = requests.get(
      'https://api.example.com/v1/docs/YOUR_PROJECT_ID',
      headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
  )
  docs = response.json()
  print(docs)
  ```
</CodeGroup>
