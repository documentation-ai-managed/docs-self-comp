## Quick Start

To begin using Self-comp, create your first documentation space:

1. Log in at `https://app.self-comp.com`
2. Click "New Space" on the dashboard
3. Enter your project name and description
4. Hit "Create"

Your documentation space is now ready. You can start adding docs and manage content right away.
